Variables Declaration

PHP variables are declared by dollar($) sign and are case sensitive

Predefined variables

These are variables accessible to all scripts also called superglobals such as $_SERVER,$_POST,$_GET etc.

Variables Scope

1.Variables in PHP only have a single scope and spans required and included files 2.variables declared inside local function have local scope inaccessible to other scripts 3.However, global keyword and $GLOBALS array can be used to access global version of the variables

Static Variables

1. Static variables unlike other variables donot lose its value even if script loses its scope 2. Static variables are resolved in compile time

Variable types

1. gettype($variable): return string => the type of the variables 2. is_array($variable): return boolean => is variable array type or not 3. is_float($variable): return boolean => is vaiable float type or not 4. is_int($variable): return boolean => is variable int type or not 5. is_object($variable): return boolean => is variable object type or not 6. is_string($variable): return boolean => is variable string type or not
