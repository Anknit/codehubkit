What is socket?

Socket uniquely identifies a network application on a computer.Socket consists of ipaddress and port number.

What is server?

Server is a computer program which provides services to other computer

What is Client?

Client is a computer who needs services from other machine(in this case server)

Working of a server

1.Bind: Program(server) tells the operating system to issue a specific port on a given network interface 2.Listen: Program(server) endlessly wait for the incoming connection on specified port 3.Accept: Accepts the incoming connection 4.Send: send something back to client 4.Close: Terminate the connection 5.repeat 2

Working of a client

1.connect: Client connect to a server socket 2.Receive: Client gets the resouces from server 2.close: Server closes the client connection
