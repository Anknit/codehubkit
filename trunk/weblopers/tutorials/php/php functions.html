PHP Functions

1. Functions are generally small piece of code that do some processing and can return some values used for reusability 2. Function names are case-insensitive 3. Function overloading and redifining or undefining is not possible in PHP 4. Functions can be recursive

Function arguements

1. By default, arguements are pass by value. 2. Arguements can be pass by refrence using & sign 3. Arguement can be array, default value, or NULL value 4. Variable length arguments list can be passed using three dots ...

Returning values

1. Function can return any type of value like string, array etc. using return word 2. Function end its execution after return
